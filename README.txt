Custom login page Customizer, Add custom logo,Custom background and more.

## Description ##
This plugin comes with a lot of feature specially for login page of wordpress.
Custom background for login page, Custom background for login form, Change all the color of login page. All of this can be done with this plugin.
Admin login page customization never being easy without this plugin.
You can change admin page background, login page background, custom logo etc.
You can see your login page by adding slug "wp-login" or "wp-admin" after your domain...


All can be done with this -->

	*   Upload custom logo to admin page
	*   Customize background image of admin page area
	*   Customize background image of admin login form area
	*   Change background color of login page
	*   Change background color of login form
	*	Change color of label for fields
	*	Change color of nav link bottom of form
	*	Change the color of submit button
	*	Change the color of submit button text
	*	Add custom css
	*	More are coming soon... 


## Installation ##

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-login-customize.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Find the settings under `Settings->Wp Login Customze`
1. Use background option tab to change background `Settings->Wp Login Customze -> Backgtround option`
1. Use Clor option tab to change colors `Settings->Wp Login Customze -> Colors and Custom css option`
